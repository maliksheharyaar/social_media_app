# Fullstack Social Blog App (MERN)

This project is a simple blog type application made with MongoDB, Express, React, NodeJS (MERN).
Features include
- Custom Signup/Sigin
- Post search by name or tags
- Google login
- Pagination
- Mobile Responsive
